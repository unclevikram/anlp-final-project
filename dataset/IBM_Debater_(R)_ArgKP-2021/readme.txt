NAME: IBM Debater(R) - ArgKP-2021

VERSION: v1

RELEASE DATE: October 22, 2021

DATASET OVERVIEW

27,519 (argument, key point) pairs labeled as matching/non-matching, for 31 controversial topics. 
For each pair, the topic and stance are also indicated.

The dataset is released under the following licensing and copyright terms:
• (c) Copyright IBM 2021. Released under Community Data License Agreement – Sharing, Version 1.0 (https://cdla.io/sharing-1-0/).

The dataset is described in the following publication: 

• Overview of the 2021 Key Point Analysis Shared Task. Roni Friedman, Lena Dankin, Yufang Hou, Ranit Aharonov, Yoav Katz and Noam Slonim. 8th Workshop on Argumentation Mining 2021.

Please cite this paper if you use the dataset.

CONTENTS

The CSV file, ArgKP-2021_dataset.csv, contains the following columns for each (argument, key point) pair:
1. topic
2. argument
3. key_point	
4. stance: 1 (pro) / -1 (con)
5. label: 1 (matching)/ 0 (non-matching)
